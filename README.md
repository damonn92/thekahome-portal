# K&A Home Company Portal

Static company portal for `thekahome.com`.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `assets/screenshots/*.jpg`

## GoDaddy upload

Upload everything in this folder to the GoDaddy hosting root, usually `public_html`.
Keep the `assets/screenshots` folder structure unchanged.

No build step is required.
